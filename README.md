#### Commonlly used  commands

Run - start a container
````
docker run <image_name>
````

ps - list containers
````
docker ps
````

show all running and exited containers
````
docker ps -a
````

STOP - stop a container
````
docker stop <image_name>
````

````
docker rm <image_name>
````
